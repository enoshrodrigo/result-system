## ğŸš€ Overview

The **BCI Result Management System** is a comprehensive platform designed to streamline the management and distribution of examination results at Benedict XVI Catholic Institute. This system bridges the gap between administrators, instructors, and students by providing a secure and efficient channel for result management and access.

The platform automates the tedious process of result compilation, verification, and publication while offering personalized portals for stakeholders to access relevant information.

## ğŸ’» Installation

### Prerequisites

- PHP 8.1 or higher
- Composer
- Node.js and npm
- MySQL

### Step 1: Clone the repository

```bash
git clone https://github.com/enoshrodrigo/result-system.git
cd result-system
```

### Step 2: Install dependencies

```bash
composer install
npm install
```

### Step 3: Environment setup

```bash
cp .env.example .env
php artisan key:generate
```

Configure your database and mail settings in the .env file:

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=root
DB_PASSWORD=

MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=your_username
MAIL_PASSWORD=your_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=
MAIL_FROM_NAME="${APP_NAME}"
```

### Step 4: Database migration and seeding

```bash
php artisan migrate
php artisan db:seed
```

### Step 5: Build assets

```bash
npm run dev
# or for production
npm run build
```

### Step 6: Launch the application

```bash
php artisan serve
```

Visit `http://localhost:8000` in your browser ğŸ‰

## ğŸ® Usage

### Admin Portal

1. Navigate to `/login` and sign in with administrator credentials
2. Use the sidebar navigation to access different modules:
   - **Dashboard**: View system statistics and recent activities
   - **Students**: Manage student profiles and enrollments
   - **Departments**: Organize academic structure
   - **Results**: Upload, verify, and publish examination results
   - **Email Logs**: Track communication with students

### Student Portal

1. Navigate to the student login page
2. Sign in with your student ID and password
3. View your personal profile and academic results
4. Download results as needed for personal records

The BCI Result Management System follows a modern, layered architecture:

### Client Layer
- **React Components**: Reusable UI elements
- **Inertia.js**: Connects the React frontend with Laravel backend
- **Tailwind CSS**: Utility-first styling with consistent design language

### Application Layer
- **Laravel Controllers**: Handle request/response flow
- **Services**: Encapsulate business logic
- **Repositories**: Abstract data access patterns

### Data Layer
- **Eloquent ORM**: Object-relational mapping for database interactions
- **MySQL Database**: Persistent storage for all system data
- **Redis Cache**: Performance optimization for frequently accessed data

### Key Architectural Patterns
- **MVC Pattern**: Clear separation of concerns
- **Repository Pattern**: Abstracted data access
- **Service Layer Pattern**: Encapsulated business logic
- **Single Page Application**: Smooth user experience with minimal page reloads
